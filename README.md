# Interactive Data Analysis and Visualization with Python

Welcome to the **Interactive Data Analysis and Visualization with Python** repository.

This repository contains course materials, notebooks, examples, exercises, and project resources designed to help learners build practical skills in **data analysis**, **data visualization**, and **interactive reporting** using Python.

In a world where data plays a central role in science, education, business, and software development, the ability to explore and present data clearly is an essential skill. This course provides a structured, hands-on path from foundational concepts to more advanced interactive visualization techniques.

## Course Overview

This course introduces learners to modern Python tools and libraries used for data analysis and visualization, including:

- **Jupyter Notebook**
- **Matplotlib**
- **Seaborn**
- **Plotly**
- **Bokeh**

The course is organized into **four modules**, each building on the previous one to create a clear and progressive learning path.

By the end of the course, learners will be able to:

- prepare and analyze data in a Jupyter environment,
- use both basic and advanced visualization techniques,
- identify patterns, distributions, and relationships in data,
- create interactive and professional visualizations suitable for presentation and sharing.

## Course Goal

The goal of this course is to help learners develop the ability to present data clearly, effectively, and interactively using Python and its most important visualization libraries.

Through practical work in Jupyter Notebook, learners will explore how to analyze data, detect patterns, visualize distributions and correlations, and build charts that are both informative and visually appealing. Special emphasis is placed on connecting analysis with presentation through interactive visualizations that can be used in business, research, reporting, and education.

## Modules

## Module 1 – Getting Started with Jupyter: Interactive Data Analysis in Python

The first module introduces learners to **Jupyter Notebook**, one of the most powerful environments for interactive data analysis. Learners will discover how to install and launch Jupyter, create their first notebook, understand the difference between cell types, and work with notebooks in a structured and efficient way.

### Topics
- First steps in the Jupyter environment
- Installing Jupyter
- Creating a first Jupyter notebook
- Working with Jupyter notebooks
- Project: First interactive analysis of a library dataset using Jupyter

### Skills developed
- Using Jupyter for creating and managing notebooks
- Presenting data analysis in notebook form
- Working with notebooks in Jupyter Notebook, JupyterLab, and Visual Studio Code

---

## Module 2 – A Different View of Data: First Steps in Visualization

The second module focuses on the foundations of graphical data presentation. Using **Matplotlib** and **Seaborn**, learners will begin creating visual representations that make data easier to understand and explain.

### Topics
- From numbers to visuals: first steps in data visualization
- Installing visualization libraries
- Drawing line charts
- Working with bar charts and pie charts
- Exporting and saving visual output
- Project: Data visualization in action

### Skills developed
- Using Python visualization libraries such as Matplotlib, Seaborn, Plotly, and Bokeh
- Creating and interpreting line charts
- Creating and interpreting bar charts
- Creating and interpreting pie charts
- Integrating visualizations into Jupyter notebooks
- Exporting visualizations to standalone files

---

## Module 3 – Visualizing Distributions and Correlations

In the third module, learners move toward a deeper understanding of data. This module introduces visual techniques used in exploratory data analysis, with a focus on understanding how values are distributed and how variables relate to one another.

### Topics
- Histograms and KDE plots
- Understanding different distribution types
- Boxplots for distribution analysis
- Correlation visualization
- Project: Using distribution and correlation visualizations to analyze library book ratings

### Skills developed
- Creating and interpreting histograms, distribution curves, and boxplots
- Understanding the distribution of data through visualization
- Creating and interpreting scatter plots and heatmaps
- Exploring relationships and correlations between variables

---

## Module 4 – Interactive Visualization

The final module adds a new dimension to the learning process: **interactive data visualization**. Learners will work with **Plotly** and **Bokeh** to create dynamic charts that support hovering, zooming, filtering, and richer exploration.

### Topics
- Interactive pie charts and line charts
- Combining histograms and boxplots with Plotly
- Interactive scatter plots for deeper correlation analysis
- Project: Creating final interactive data analysis reports

### Skills developed
- Using Plotly and Bokeh for interactive chart creation
- Building interactive histograms, pie charts, line charts, boxplots, and scatter plots
- Creating interactive web-based reports that document analytical results
